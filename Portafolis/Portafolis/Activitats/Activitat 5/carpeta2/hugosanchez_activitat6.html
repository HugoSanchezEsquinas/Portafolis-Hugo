Llista de definició

Objectiu: Fer una llista amb termes relacionats amb el món web i la seva definició.

HTML
Llenguatge de marcat que estructura el contingut d'una pàgina web.
CSS
Llenguatge d'estils que defineix l'aparença visual d'una pàgina web.
JavaScript
Llenguatge de programació que permet afegir interactivitat a les pàgines web.
Navegador
Aplicació que permet visualitzar pàgines web, com ara Chrome o Firefox.
Pàgina web
Document digital accessible a través d'internet mitjançant un navegador.
E-commerce
Comerç electrònic realitzat a través d'internet.
W3C
Organització que estableix els estàndards del web.
Internet
Xarxa mundial que connecta milions de dispositius i usuaris.
